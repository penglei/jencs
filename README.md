jencs
=====

clearsilver template engine with debugger

##usage

1.安装依赖

    npm -g install express
    npm -g install socket.io
    npm -g install nomnom

2.使用示例

    git clone https://github.com/penglei/jencs
    cd jencs
    node test.js --enable-debugger

使用chrome浏览器访问下面的地址:

   http://127.0.0.1:10080/
